Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


#This Program Converts Celsius to Fahrenheit
#Written by Charles Slack

#define methods
def convert(c)
    return (c * 9/5) + 32
end

# ask for user input
puts "Please enter the degrees in Celcius, that you would like to convert to Fahrenheit:"
temp = gets.chomp

# convert string to float
cel = temp.to_f

# store conversion answer to varible
f = convert(cel)

# convert to string
output = temp + " degrees Celsius is " + f.to_s + " degrees Fahrenheit"

#write answer to terminal
puts output

Ordinal Challenge

This ruby program will convert a plain number to the ordinal of the number. So for example, if the user enters 2, it will display 2nd, if the user enters 3, it will display 3rd, etc.

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 


#This program solves for FOOBAR
#Written by charles slack

#user input
puts "How many items do you want to see?"
var= gets.to_i

 #do a loop        
var.times do |a|
    var += 1 
    foo = a % 3
    bar = a % 5
    if foo == 0 && bar == 0
      print " foobar "
    elsif foo == 0
      print " foo"
    elsif bar == 0
      print " bar "
    else
     print " #{a} "
    end  
end
